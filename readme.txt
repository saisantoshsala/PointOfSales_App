Point of Sales Application
-------------------------------------------------------------------------------------------------

This application is designed to add items to the existing list.

Initially, the user can login with Username = "SaiSantosh" and Password="Matrix". If the credential are correct, he will be given further privilege.

Else, An error message is shown.

Then, User can add the items that already exist to the cart. But the newly added items and respective prices are shown in datagridview and items view.

Several Validations are also included like automatic data reset when navigating from point to point.

The cost of the products added to the cart are shown in the receipt view which you can also reset.

User can exit the application by clicking Exit button.

Before exiting, he has an option to export the receipt into Xml file that can be found in Debug folder of Bin.
